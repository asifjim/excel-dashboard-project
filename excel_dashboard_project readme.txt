📌 Project Overview

This project is an interactive Excel Sales Dashboard designed to analyze sales performance across different regions, product categories, and years.
The dashboard allows users to quickly identify key trends and insights using dynamic filters and visualizations.

🎯 Objectives

Analyze overall sales performance

Compare sales across regions and categories

Track yearly sales trends

Provide an easy-to-understand visual summary of the data

🛠 Tools & Features Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers for interactive filtering

Data cleaning and formatting

KPI indicators

📂 Dataset

The dataset contains information about:

Sales revenue

Product category

Region

Year

The dataset used in this project is included in the repository.

📈 Key Insights

Some insights derived from the dashboard:

Certain regions contribute significantly more to total revenue

Sales trends fluctuate across different years

Product categories show varying levels of performance

🚀 How to Use

Download the Excel file from this repository.

Open the file in Microsoft Excel.

Use the slicers to filter the data by region, year, or category.

The charts will update automatically.
